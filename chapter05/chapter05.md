# モジュールとパッケージ
- モジュールはバージョニングするまとまり
- パッケージは特定のディレクトリ

# Go Moduleの環境変数
`GO111MODULE`という環境変数で挙動が変わるが、1.16からはデフォルトで`ON`になっている。

# 依存先コードでデバッグする

- `go get`で取得したコードは `GOMODCACHE` で指定されたディレクトリに入る。
- `go mod vendor`でモジュール内に`vendor`ディレクトリを作成する。
  - `go mod`はvendorディレクトリがある場合それを読む。
